1. Укажите правильные утверждения 
            Программы-вирусы специально предназначены для того, чтобы нарушать нормальную работу компьютерных систем 
            Компьютерный вирус - это набор команд, который производит и распространяет свои копии в компьютерных системах и/или компьютерных сетях 
            Компьютерный вирус - это набор команд, который преднамеренно выполняет некоторые действия, нежелательные для законных пользователей системы
