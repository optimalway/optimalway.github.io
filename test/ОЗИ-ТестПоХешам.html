1. Хэш-функцией называется 
            односторонняя функция, предназначенная для получения дайджеста или "отпечатков пальцев" файла, сообщения или некоторого блока данных 
            
        2. Хэш-код создается функцией Н, такой что h = H (M), где 
            М является сообщением произвольной длины и h является хэш-кодом фиксированной длины 
        
        3. Хэш-функции 
            нельзя использовать для шифрования 
            нельзя использовать для шифрования со слишком длинным ключом 
        
        4. Укажите правильные утверждения о свойствах хеш-функции Н. h = H(M) 
            Хэш-функция Н должна применяться к блоку данных любой длины 
            Н (М) относительно легко (за полиномиальное время) вычисляется для любого значения М 
            Для любого данного значения хэш-кода h вычислительно невозможно найти M такое, что Н (M) = h 
        
        5. Сообщение зашифровано аффинным шифром Цезаря по формуле ci = (mi+5)*4 mod 26. Укажите формулу для расшифровки 
            Хэш-функция Н создает выход фиксированной длины 
            Для любого данного х вычислительно невозможно найти y != x, что H (y) = H (x) 
        
        6. Понятию 'вычислительно легко' соответствует 
            Линейная
            сложность вычислений 
            Полиномиальная сложность вычислений 
            Логарифмическая сложность вычислений 
        
        7. Понятию 'вычислительно невозможно' соответствует 
            Экспоненциальная сложность вычислений 
        
        8. Хэш-функция md5 
            создает хэш-код длиной 16 байт 
            создает хэш-код длиной 128 бит 
        
        9. Хэш-функция sha256 
            создает хэш-код длиной 32 байта 
            создает хэш-код длиной 256 бит
